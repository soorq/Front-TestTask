## Getting Started

To get a local copy up and running follow these simple example steps.

### Prerequisites

This is a example for installation project and start in local

* npm
  ```sh
  npm install npm@latest -g
  ```

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/soorq/Front-TestTask.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Enter your enviroments in .env file
   ```sh
      > NEXTAUTH_SECRET=YOUR SECRET
      > EXTAUTH_URL=YOUR CLIENT APP URL
      > NEXT_PUBLIC_API_URL=YOUR API URL
      > NEXT_PUBLIC_APP_URL=YOUR CLIENT APP URL
   ```

## Usage

### Start dev

```bash
> npm run dev
```

```bash
> yarn dev
```

```bash
> pnpm dev
```

### Start prod version

```bash
> npm run build
> npm run start
```

```bash
> yarn build
> yarn start
```

```bash
> pnpm build
> pnpm start
```
